## Steps to Run Code

1. Copy the repository
```
git clone --recursive https://github.com/Atharva-Paralikar/A-star-non-holonomic-constraints.git
```
2. Source the repository 
```
cd ~/A-star-non-holonomic-constraints
```
3. Run the package 
```
python3 astar.py
```
